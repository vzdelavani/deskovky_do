# Výběr deskových her pro knihovny (testovací verze)

Jednoduchý konzolový nástroj pro knihovníky, který pomáhá
doporučit deskové hry podle:
- věkové skupiny
- délky hraní
- stylu hry
- tématu

## Jak spustit
Je potřeba mít nainstalovanou Javu (verze 17+).

1. Stáhni soubor `GameSelectorConsole.java`
2. Otevři terminál ve složce se souborem
3. Spusť:

```bash
javac GameSelectorConsole.java
java GameSelectorConsole
